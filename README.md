# Essence Coffee Manual Project

## 1. Project Goal

To provide a mobile-friendly digital manual for customers via QR code printed on Essence Coffee products.

## 3. How to Launch (Enable Website)

1. Go to your GitHub repository: [Essence-Coffee-Manual](https://github.com/coffeelike5282/Essence-Coffee-Manual)
2. Click **Settings** (top right tab).
3. Click **Pages** (left sidebar).
4. Under **Source**, select **main** branch and click **Save**.
5. Wait a few minutes, and your site will be live!

## 4. How to Add Manual Images

1. Prepare your manual pages as images (JPG or PNG).
2. Name them sequentially (e.g., `manual_01.jpg`, `manual_02.jpg`).
3. Upload them to this repository.
4. Edit `index.html` to include them:

   ```html
   <img src="manual_01.jpg" class="manual-page">
   <img src="manual_02.jpg" class="manual-page">
   ```

5. Commit changes.
