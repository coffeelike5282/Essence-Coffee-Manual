# Essence Coffee Manual Project

## Project Goal

To provide a mobile-friendly digital manual for customers via QR code printed on espresso coffee products.

## How it works

1. **QR Code**: Customer scans the QR code on the product.
2. **Landing Page**: Opens this mobile-optimized web page.
3. **Manual**: User can view the manual (PDF or Web verstion) easily.

## Tech Stack

- HTML5 / CSS3 (Mobile First)
- GitHub Pages (Hosting)
